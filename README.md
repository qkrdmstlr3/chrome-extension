## Developer

### How to use? (EN)

1. install [nodeJS](https://nodejs.org/ko/)
2. clone this repo
3. execute command
  ```
  npm install
  npm run build
  ```
4. open chrome extension web page (chrome://extensions/)
5. toggle develop mode (located in the upper right)
6. click `Load the unzipped extension` button
7. select `build` folder
